Angular Failed Requests
=======================

Angular Module to manage your failed requests

Installation
------------
Add "localStorage" and "failedRequestsManager" to your app modules list:
angular.module('MyApp', [
        'MyApp.controllers',
        'MyApp.services',
        'MyApp.filters',
        'MyApp.directives',
        'localStorage',
        'failedRequestsHandler'
        ])
        
        ....
        
        ;
